基于kintohub免费空间部署v2ray

喵ฅ仙 | BLOG V2Ray

提醒： 免费空间请不要滥用！！！ 本项目支持部署kintohub和heroku两个空间，下面是部署到kintohub方法，部署到heroku方法请见：[Deploy to Heroku]> (https://github.com/yeahwu/kinto/blob/master/v2-heroku.md) 利用kintohub免费空间部署v2ray和tor，部署成功后，可用v2ray客户端直接访问tor网络，比如暗网.onion，v2ray.json文件中设置了路由分流了tor网络，非tor流量不受影响。

# 注册 请翻墙注册 国内ip被kintohub封了

区域选取北美或者欧洲，亚洲凉了

# 部署服务端

  1. 点开 https://app.kintohub.com/ 新建一个APP，点击 Create Service ,然后创建 Web App 如图：

  1. Repository 填上git链接和默认分支master

  1. Build Settings 文件名填写Dockerfile，端口填上8888，填写如下：

最后点击右上角 Deploy，部署完成，会生成一个链接，点击链接，如果显示Bad Request，即为成功。
