Docker部署运行springboot项目，并使用Dockerfile制作镜像

喵ฅ仙 | BLOG Dockerfile

# 将springboot项目打包 jar包

# 创建Dockerfile文件

编写Dockerfile

FROM java:8
COPY *.jar /springboot.jar
CMD ["--server.port=5050"]
EXPOSE 5050
ENTRYPOINT ["java","-jar","/springboot.jar"]

# 小知识点

ENTRYPOINT拼接CMD 实际上运行的脚本为

java -jar springboot.jar --server.port=5050
